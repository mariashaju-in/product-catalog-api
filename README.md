# Product Catalog API

This is a simple backend project I built using Spring Boot to understand how REST APIs are designed and structured in real applications.

The project allows basic product management operations like creating, viewing, updating, and deleting products.

---

## What this project does

- Create new products  
- View all products (with pagination)  
- Get product by ID  
- Update product details  
- Delete products  
- Basic validation and error handling  

---

## Tech Stack

- Java 17  
- Spring Boot  
- Spring Data JPA  
- H2 Database  
- Maven  

---

## Why I built this

I wanted to get hands-on experience with backend development instead of just learning concepts. This project helped me understand how different layers in a Spring Boot application work together and how APIs interact with a database.

---

## What I learned

- Structuring a Spring Boot project using controller, service, and repository layers  
- Designing REST APIs and handling requests/responses  
- Implementing pagination and sorting  
- Handling exceptions in a centralized way  
- Using H2 database for quick development and testing  

---
